
Цей архів містить простий статичний сайт про ромашки в полі.

Файли:
- index.html
- style.css

Як опублікувати сайт (два прості способи):

1) GitHub Pages (безкоштовно):
   - Створи новий репозиторій на GitHub.
   - Завантаж (push) файли index.html і style.css в гілку main.
   - У налаштуваннях репозиторію увімкни Pages і обери гілку main → / (root).
   - Через кілька хвилин сайт буде доступний за посиланням: https://<твій_юзернейм>.github.io/<ім'я_репо>/

2) Vercel (безкоштовно для простих сайтів):
   - Зареєструйся на vercel.com і підключи репозиторій GitHub.
   - Створи новий проєкт і вкажи, що це статичний проект.
   - Vercel автоматично задеплоїть сайт і дасть URL виду https://<назва>-<хеш>.vercel.app

Якщо хочеш, я можу сам(а) підготувати репозиторій-ready zip і допомогти з деплоєм — скажи, яку платформу обираєш: GitHub Pages чи Vercel.
